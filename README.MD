                            Ejercicio 1


En este ejercicio lo que realizaremos es un juego de bombas. Pediremos a nuestros usuarios que nos den unas dimensiones para poder crear el tablero. Con el tablero creado por las dimensiones dadas por los usuarios, pondremos valores de forma aleatoria que ellos no sabran. Tendran un menu para poder ver la matriz y saber cuales les quedan ya que con cada bomba la columna y fila se pondran en valor 0.
Una vez haya llegado al final, o durante el juego, podra ver el ranking de bombas que ha realizado para ver cual a hecho mas daño.
